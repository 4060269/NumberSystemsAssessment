Decimal Conversion

Converting from Decimal to Binary (and other number systems) is needed for various reasons in Computer Science.

This page will convert the decimal number you enter into the binary (base 2) and the hexadecimal (base 16). This will be done via various functions.

In the HTML file, there will eventually be a number of functions (modules).

startConversionProcess() will run when the user presses the button below. This function will prompt the user for the binary number through a prompt() command.

This function will call the other functions, listed below.

convertToBinary() - converts the number entered in Decimal (base 10) in to binary and updates the relevant HTML tag.
convertToHexadecimal(decimalNumberToConvert) - converts decimalNumberToConvert into Hexadecimal (base 16) and will return the value back to startConversionProcess to update the relevant HTML.
